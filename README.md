# GPSDog
Arduino 3G / GPS watch guard for movable objects

## SMS commands

- ```INIT password number ON/OFF```
- ```RESET password```
- ```STATUS```
- ```INTERVAL min```
- ```STORE idx ADD number ON/OFF```
- ```STORE idx DEL```
- ```STORE idx SHOW```
- ```WATCH ON/OFF/?```
- ```PROTECT ON/OFF/?```
- ```ALARM ON/OFF/?```
- ```STOP```
